The purpose of this project is to demonstrate java fundamentals. In this case, this file demonstrates object instatiation and accessing objects and their data between the classes Apartments and TestApartments.


---TEST APARTMENTS---
Apartment objects are created with their respective bed, bath, and costs. 
The information is then displayed by using get methods.


---APARTMENTS---  
The Apartments class constructor accepts apartment number, number of beds, number of baths, and monthly rent. The constructor call the setter methods and feeds them their respective information. 


How to use: 
This file's purpose is to provide an example sandbox for designing simple classes and observing data flow between them. Adjust source code to the desired function and test.
